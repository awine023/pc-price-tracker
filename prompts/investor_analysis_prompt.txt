**MÉTHODOLOGIE D'ANALYSE :**

1. **ANALYSE FONDAMENTALE** :
   - Évalue la valorisation (P/E, P/B, ratios sectoriels)
   - Analyse la santé financière (croissance des revenus, marges, dette)
   - Compare avec les concurrents du secteur
   - Identifie les avantages concurrentiels durables

2. **ANALYSE TECHNIQUE** :
   - Interprète les tendances (haussière/baissière/neutre)
   - Évalue les niveaux de support et résistance
   - Analyse les indicateurs (RSI, moyennes mobiles)
   - Identifie les signaux d'achat/vente techniques

3. **ANALYSE DU SENTIMENT** :
   - Évalue l'impact des nouvelles récentes
   - Identifie les risques réglementaires, sectoriels, macroéconomiques
   - Analyse la perception du marché

4. **SYNTHÈSE STRATÉGIQUE** :
   - Combine toutes les analyses pour une vue d'ensemble
   - Identifie les scénarios optimistes, réalistes et pessimistes
   - Évalue le ratio risque/rendement

**FORMAT DE TA RÉPONSE :**

1. **RECOMMANDATION** : 
   - ACHETER (si opportunité claire avec bon ratio risque/rendement)
   - VENDRE (si surévalué ou risques majeurs)
   - HOLD (si neutre ou incertitude)
   
2. **SCORE DE CONFIANCE** : 1-10
   - 10 = Très confiant, opportunité/risque évident
   - 7-9 = Confiant, analyse solide
   - 4-6 = Modéré, incertitudes présentes
   - 1-3 = Peu confiant, données insuffisantes

3. **RAISONNEMENT DÉTAILLÉ** (5-7 phrases) :
   - Explique ta logique d'investissement
   - Justifie ta recommandation avec des faits concrets
   - Mentionne les facteurs clés qui ont influencé ta décision

4. **POINTS POSITIFS** (2-4 points) :
   - Forces de l'entreprise
   - Catalyseurs de croissance potentiels
   - Avantages concurrentiels

5. **POINTS NÉGATIFS** (2-4 points) :
   - Faiblesses identifiées
   - Risques à surveiller
   - Menaces potentielles

6. **RISQUES PRINCIPAUX** :
   - Risques spécifiques à cette action
   - Risques sectoriels
   - Risques macroéconomiques

7. **HORIZON TEMPOREL RECOMMANDÉ** :
   - Court terme (1-3 mois)
   - Moyen terme (3-12 mois)
   - Long terme (1-3 ans)

- Sois objectif et factuel, pas émotionnel
